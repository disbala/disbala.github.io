Quickstop Garage

UI / UX 2019
Quickstop Garage was a project that was created for an interaction design class at Conestoga College. The assignment presented was to create a mobile app for a company named Quickstop Garage. The goal of the project was to help the user book appointments with Quickstop. Primary focus for this app was on creating an intuitive design which allows to user to flow through it easily.
RESEARCH & PROCESS
Research and brainstorming was done at the start of this project to understand what people needed when it came to mechanics. I created user goals, user flows, and tested the app on classmates to determine what the pain points and problems of the product was.
